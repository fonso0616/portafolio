PG NEWS - Proyecto personal


PG News es una aplicación web para mostrar, agregar y eliminar noticias de diferentes categorías. Los usuarios pueden ver las noticias disponibles y también añadir nuevas noticias a la base de datos.

XpertCode - Pagina web para pasantía


Desarrollé una página web para la empresa XpertCode, ésta consiste en un portal mediante el cual, los usuarios pueden ingresar a un sistema de gestión de empleados, donde pueden ver sus datos personales, así como también las vacaciones que tienen disponibles y las que ya han utilizado.

AURALIS - Proyecto final para la universidad


Participé activamente en el desarrollo de la app AURALIS, una aplicación de relajación y meditación, que permite a los usuarios acceder a una variedad de sonidos y meditaciones guiadas para mejorar su bienestar mental. La app está diseñada para ser intuitiva y fácil de usar, con un enfoque en la experiencia del usuario.
